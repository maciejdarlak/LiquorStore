# LiquorStore

ASP.NET.Core 3.0

## Overview

> This is a website that was built in C # ASP.NET Core.
> It is a web application that allows you to administer an alcohol store and buy its products.



## Frameworks & Dependencies
- Entity Framework Core
- ASP.NET Core
- Bootstrap
- .NET tag helpers
